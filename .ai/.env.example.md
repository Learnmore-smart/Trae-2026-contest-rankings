# .env.example

> Last updated: 2026-07-01 | Protection: STANDARD

## Purpose

Documents non-secret environment variables and placeholders required to configure local and deployed runs.

## What It Does

- Lists provider, Firebase/Data Connect, admin, cron, scraper, matcher, and judge tuning variables.
- Keeps real secrets out of the repository.

## Public API

| Name | Type | Description |
|------|------|-------------|
| `TRAE_JUDGE_CONCURRENCY` | env var | Optional bounded parallelism for topic judging. |
