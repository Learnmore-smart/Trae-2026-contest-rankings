# .env.example

> Last updated: 2026-07-04 | Protection: STANDARD

## Purpose

Documents non-secret environment variables and placeholders required to configure local and deployed runs.

## What It Does

- Lists provider, Firebase/Data Connect, admin, cron, scraper, matcher, and judge tuning variables.
- Documents the per-key 40 rpm NVIDIA default, optional second NVIDIA key, and unlimited rate-limit retry default.
- Keeps real secrets out of the repository.

## Public API

| Name | Type | Description |
|------|------|-------------|
| `TRAE_JUDGE_CONCURRENCY` | env var | Optional bounded parallelism for topic judging. |
| `AI_RPM_LIMIT` | env var | Per-key LLM request-start limit; default example is 40 rpm. |
| `AI_MAX_RATE_LIMIT_RETRIES` | env var | Rate-limit retry budget; `0` means unlimited retry until throttle clears. |
| `NVIDIA_API_KEY_2` | env var | Optional second NVIDIA key; additional numbered keys are supported by config. |
| `TRAE_MAX_JUDGE_PER_RUN` | env var | Per-run cap for selected judge topics; default example is 4000. |
