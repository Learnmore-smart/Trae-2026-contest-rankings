# app/trae-contest-2026/i18n.tsx

> Last updated: 2026-06-29 | Protection: STANDARD

## Purpose

Provides lightweight route-local language state and a reusable settings language dropdown.

## What It Does

- Defaults contest pages to Chinese.
- Persists English or Chinese preference in `localStorage`.
- Exposes a compact settings dropdown with smooth open/close transitions.

## Public API

| Name | Type | Description |
|------|------|-------------|
| `useContestLanguage` | hook | Reads and writes the persisted contest language. |
| `LanguageSettings` | component | Renders a settings button and zh/en dropdown. |

## Dependencies

- External: React hooks and `lucide-react` icons.

## Agent Decisions / Thoughts

- 2026-06-29 Codex: Keep i18n route-local because this project does not have a broader i18n router or dictionary system, and the contest pages are the only public website surface.
- 2026-06-29 Codex: Preserve Chinese first-load behavior by initializing state to `zh` before reading `localStorage`.
- 2026-06-29 Codex: Implemented outside-click and Escape handling so the dropdown behaves like a normal settings menu.
- 2026-06-29 Codex: Keep language switching as a custom dropdown, not a native select, so every dropdown surface is styled consistently.
